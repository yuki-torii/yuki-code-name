# ⛩  变量名称规则

## 基本规则

 - 必须使用驼峰
 - Class必须开头大写
 
## 自查表
| 描述         	| 名称   	| 说明                           	|
|--------------	|--------	|--------------------------------	|
| 用户 	| user   	| 不要用people、person、human    	|
| 开关 	| is*    	| 例如isShow、isOpen、不要用has* 	|
| 微信         	| wechat 	| 不要用weixin和wx               	|
| 消息  | msg/message | 函数中不要用全称，组件用全称 |
